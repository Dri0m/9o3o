Frequently Asked Questions
What's 9o3o?
9o3o is an experimental web version of the Flashpoint Archive, developed as a side project by some of its devs. It is heavily inspired by z0r.de and booru imageboards. Its source code can be found here.
How does it work?
9o3o uses the open-source emulators Ruffle and X_ITE to enable the playback of Flash, VRML, and X3D content in modern browsers. It also employs a novel JavaScript redirector to intercept network requests in a similar fashion to Flashpoint's own redirector.
An entry isn't working!
If the problem has to do with the emulator's playback of the entry, consider opening an issue in that emulator's GitHub repository:
  • Ruffle issue tracker
  • X_ITE issue tracker
Otherwise, open an issue in 9o3o's own issue tracker.
